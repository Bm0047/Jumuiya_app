Maria Magdalena Community App - Starter Scaffold

What's included:

- Vite + React + TypeScript starter
- Placeholder Firebase dependency in package.json
- Basic App and styles

Next steps:

1. Run `npm install` to install dependencies.
2. Create a Firebase project, enable Firestore and Authentication, and copy the config into `src/firebaseConfig.ts` (create the file).
3. Start dev server with `npm run dev`.
